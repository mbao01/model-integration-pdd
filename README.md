## Ruby utility class to prioritize puzzles

#### Dependencies

- [Python](https://www.python.org/) >= 3.6
- [Ruby](https://www.ruby-lang.org/en/) >= 2.6

#### Setup

- install dependencies

```bash
$ bundle install
$ pip install -r ./python/requirements.txt
```

#### Run

```bash
$ ruby model.rb
```
